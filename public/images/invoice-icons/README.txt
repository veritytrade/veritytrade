Invoice header icons - save your icon files here with these exact names:

  location.png   - Location/address icon (or location.svg, location.jpg)
  email.png      - Email icon (or email.svg, email.jpg)
  phone.png      - Phone icon (or phone.svg, phone.jpg)

Supported formats: PNG, SVG, JPG (checked in that order)
Recommended size: 14x14 to 24x24 pixels

If a file is missing, the invoice uses the built-in SVG icon for that slot.
